Defining a Grid

  • To define a Grid use display:grid on the parent element. You can then create a grid using the grid-template-columns and grid-template-rows properties.
  • grid-gap property is used to create a gap between the grid columns and rows. This property is a shorthand for grid-column-gap and grid-row-gap that can be used to set these values individually.
  • All direct children of the parent become grid items and they are auto-placed in the grid cells. Extra rows are auto-created as needed.
