Placing elements using areas

  • We can create named areas on the grid to put content into using grid-template-areas to describe where on the Grid elements should sit.
  • Assign an element to a Grid Area using the grid-area property.
  • Repeating the name of an area indicates that the area spans that grid track. Using a . indicates an empty track. Such as the very first cell on this grid.
