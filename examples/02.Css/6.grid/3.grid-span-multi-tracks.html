Spanning elements on multiple tracks

  • Positioning the elements (six divs with a class from a to f) using line-based placement properties.
  • The column is decided using grid-column by specifying the column start line and end line.
  • The row is decided using grid-row by specifying the row start line and end line.
  • If an item will only span one grid cell you may omit the / and the end value.
